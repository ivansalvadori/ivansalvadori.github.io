TÍTULO: Infraestruturas de software para geração e publicação de dados conectados na Web


RESUMO: 
Empresas, governos e até mesmo pessoas comuns produzem e publicam grandes quantidades de dados na Web.
Esse fenômeno alavancou o interesse em áreas como ciência de dados, que busca extrair conhecimento e valor a partir de análises realizadas em um determinado conjunto de dados.
Muitos observadores, entretanto, apontam que extrair conhecimento de dados requer técnicas e ferramentas adequadas, dentre elas a doação de métodos de integração de dados.
Pesquisas recentes apontam que a semântica dos dados é fundamental para fomentar soluções de integração de dados.
No entanto, faltam soluções para geração, manutenção e publicação de dados que sigam as práticas recomendadas.
Nesta palestra, serão apresentadas infraestruturas de software para manipulação de dados conectados, além de abordar o uso de Web semântica para o desenvolvimento de sistemas mais flexíveis e resilientes.
